# Audio Switcher

Windows で **音声出力デバイスをホットキーで瞬時に切り替える**ためのツールです。

バックグラウンドで常駐し、Ctrl + Alt + 数字 などのキー操作で再生デバイスを変更できます。

audioswitcher.exeを実行するだけですぐに使うことができます。

---

## スタートアップ登録（PC 起動時に自動で起動させる）

毎回手動で起動したくない場合は、**スタートアップに登録**してください。

### 手順

1. `Win + R` を押す
2. `shell:startup` と入力して Enter
3. 開いたフォルダに`AudioSwitcher.exe` の **ショートカット** を置く

---

## 終了方法

タスクトレイには表示されません。

終了したい場合は以下のいずれかを行ってください。

- タスクマネージャーを開く
- `AudioSwitcher.exe` を選択
- 「タスクの終了」をクリック

---

## 注意事項

- Windows 専用ツールです
- 他のアプリとホットキーが競合すると動作しない場合があります
- 音声デバイス名は PC 環境によって異なります
